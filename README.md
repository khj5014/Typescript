# Type-script
 타입스크립트 배우기


환경 설치
1. node.js 설치
2. 터미널 선택
3. ↓ 코드 입력 ↓
npm init                        // package.json 생성
tsc init                        // tsconfig.json 생성
npm i -g typescript             // 타입스크립트 설치 '-g' <- 전역설치
npm i ts-node --save-dev        // 타입스크립트를 노드에서 실행 시키기 위해 런타임 설치
npm i nodemon --save-dev        // 자동화

VSCODE 확장 프로그램 - TypeScript Debugger 설치
